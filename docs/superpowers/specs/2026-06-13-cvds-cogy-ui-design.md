# CVDS COGY UI 设计说明

## 目标

在不改变检测、ROI、配置和发布逻辑的前提下，将 CVDS C++ Detector 2.0 重做为“监控画面优先”的深色工业值守台，并统一使用用户提供的 COGY 氪技品牌图。

## 设计方向

- 风格：Night Operations Industrial Console，深色、紧凑、克制。
- 主色：COGY 蓝 `#2F88F5`，仅用于焦点、选中和主操作。
- 背景：`#0B1118`；面板：`#111B25`；抬升层：`#172431`；边框：`#263746`。
- 文字：主文字 `#F3F7FA`；次要文字 `#8FA5B8`。
- 状态：正常 `#36C98F`；警告 `#FFB84D`；堵包/错误 `#F25555`。
- 圆角保持 4-6px，不使用渐变、玻璃、厚阴影和装饰动画。

## 布局

1. 顶部增加 56px 品牌状态栏：左侧 COGY Logo 与产品名，右侧显示版本和“系统就绪”状态。
2. 主体保留水平分隔布局，左侧设置栏默认约 312px，可拖动，右侧监控区占剩余空间。
3. 右侧顶部四个 KPI 单行展示；中间实时画面获得最大伸展空间。
4. 区域状态表固定为紧凑高度；运行日志继续默认隐藏，从底部按钮展开。
5. 窗口默认最大化，最低 960×600 下仍可使用。

## 品牌资源

- 原图 `COGY氪技.jpg` 保持品牌比例，不使用 Stitch 自动生成的替代 Logo。
- 完整 Logo 用于顶部品牌栏。
- 左侧图形标志输出透明 PNG 和多尺寸 ICO，用于窗口、任务栏、EXE 和安装程序。
- 品牌资源通过 Qt Resource System 嵌入程序，发布目录不依赖外部图片。

## 行为约束

- 不修改检测参数、模型读取、worker 协议、ROI 数据和看板计算。
- 堵包闪烁只改变边框与状态强调，不把整个界面染成红色。
- 控件禁用、焦点、选中和悬停状态必须清晰。
- 所有原有按钮和字段继续可访问，左侧通过滚动容器容纳小屏幕。

## 验收

- 源码结构测试覆盖品牌资源、顶部栏、A 版配色和监控画面优先布局。
- C++ Release 编译通过，隐藏启动通过。
- 正式 EXE、worker 和安装包签名状态为 Valid。
- 打开正式程序进行截图核对，Logo、布局、看板和日志折叠均正确。
