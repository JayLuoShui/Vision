# 部署说明

运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\apps\DatasetAssistant\packaging\build_release.ps1
```

输出：

- `dist/DatasetAssistant/`
- `dist_installer/DatasetAssistant_Setup_<version>.exe`

发布包包含 Qt、OpenCV、ONNX Runtime DLL、文档和版本文件。

## 标注同步可视化校验

开发机可运行：

```powershell
.\build\DatasetAssistant\visual_annotation_transform.exe .\build\annotation_transform_visual
```

然后打开 `build/annotation_transform_visual/`，检查 bbox、多边形和 mask 在 resize、letterbox、crop、tiling、flip、90/180/270 rotate 后是否仍贴合目标。
