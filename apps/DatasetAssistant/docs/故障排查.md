# 故障排查

- 无法启动：检查安装目录是否包含 Qt `platforms/qwindows.dll`。
- 视频打不开：检查 OpenCV videoio/ffmpeg DLL 是否随包发布。
- ONNX GPU 不可用：改用自动或 CPU，或安装匹配 NVIDIA 驱动与 CUDA 运行包。
- 输出失败：不要把输出目录设在 Program Files，使用默认 AppData 目录。
- 单张图片失败：查看日志目录中的 `failed_items.csv`。
- 构建机提示 Application Control 或 Device Guard：这是本机安全策略拦截新生成 exe；发布脚本会先跑单元测试，若只是在 `--diagnose` 烟测阶段被拦截，会继续生成安装包。
