# CVDS 多 ROI 手工验收

## 预览与 ROI

- [ ] 本地视频点击“应用本地视频”后只显示首帧，不自动播放检测。
- [ ] 海康视频流点击“应用视频流”后能实时预览。
- [ ] 多路视频源能合成宫格画面。
- [ ] 点击某个子画面后，绘制 ROI 归属到该路相机。
- [ ] 新增、重命名、删除区域后，画面和区域表同步更新。
- [ ] 未闭合 ROI 时不误显示区域运行数据。

## 检测与统计

- [ ] 点击“开始检测”后，预览线程先释放，检测线程接管视频源。
- [ ] 多路检测输出到 `camera_1`、`camera_2` 等子目录。
- [ ] 顶部累计包裹跟随当前计数口径。
- [ ] 区域统计详情显示所有区域明细。
- [ ] 包裹中心点进入 ROI 后计数，离开后不重复计同一轨迹。
- [ ] 目标框在 ROI 外为黄色，进入或压线后为绿色。

## 堵包与告警

- [ ] 区域内有包裹且速度低于阈值时触发堵包。
- [ ] 堵包时顶部状态、KPI 卡片、视频边框和区域行红色闪烁。
- [ ] 任一路任一区域堵包时，全局红色告警保持有效。
- [ ] 堵包解除后红框恢复正常。
- [ ] `jam_signals.jsonl` 写入 `IO_JAM_ON` 和 `IO_JAM_OFF`。

## 发布包

- [ ] 发布目录不包含 Python、conda、Torch、Ultralytics、worker、`.pt`、`.onnx`。
- [ ] 发布目录包含 `openvino_ir_frontend.dll`。
- [ ] OpenVINO `.xml + .bin` 能加载。
- [ ] 带 TensorRT 构建时，`.engine/.plan` 能加载。
