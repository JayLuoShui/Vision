# CVDS AI 辅助 YOLO 标注工具 v2.3 用户使用说明

启动 `CVDS_Annotation_Tool_v2.3.exe` 后，先选择图片目录和输出目录。默认输出写入 `%LOCALAPPDATA%/CVDS/AnnotationTool/datasets/cvds_annotation_yolo`。

基础功能不依赖 AI 环境：可以打开图片、画检测框、画分割 polygon、标注缺陷、Undo/Redo、保存 YOLO txt、保存 defects JSON。

AI 和 SAM 是增强功能。点击“环境自检”可以查看当前发布包是否包含 Ultralytics、Torch、CUDA 和权重。
