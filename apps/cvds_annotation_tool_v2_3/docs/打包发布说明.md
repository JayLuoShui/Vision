# 打包发布说明

基础版打包：

```powershell
powershell -ExecutionPolicy Bypass -File .\apps\cvds_annotation_tool_v2_3\packaging\build_release.ps1
```

包含 AI 依赖版：

```powershell
powershell -ExecutionPolicy Bypass -File .\apps\cvds_annotation_tool_v2_3\packaging\build_release.ps1 -IncludeAI
```

发布结果：

- `dist/CVDS_Annotation_Tool_v2.3/CVDS_Annotation_Tool_v2.3.exe`
- `dist/CVDS_Annotation_Tool_v2.3.zip`
