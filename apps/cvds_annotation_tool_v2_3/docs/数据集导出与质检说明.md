# 数据集导出与质检说明

“数据集质检”会扫描当前输出目录，统计图片数、标签数、空标签、缺失标签、孤儿标签、非法 YOLO 行、越界坐标、异常缺陷和类别分布。

输出文件：

- `reports/dataset_quality_yyyyMMdd_HHmmss.json`
- `reports/class_distribution.csv`

“导出数据集”不会修改当前工作数据集，会生成标准 YOLO 目录：

- `images/train`
- `images/val`
- `labels/train`
- `labels/val`
- `defects/train`
- `defects/val`
- `data.yaml`
